# DocsHub

> 간편하게 사용할 수 있는 팀 별 문서 저장소
> [백엔드](https://github.com/ashdown0069/docshub-backend)

## ✨ 주요 기능

- **파일브라우저**: 폴더 생성 및 파일 업로드, 삭제 이동 등 파일관리 기능과 드래그 앤 드랍을 통한 간편한 이동구현
- **저장소**: 여러개의 목적별 저장소 생성 가능
- **권한 기능**: 어드민이 사용자의 다운로드, 업로드, 수정, 삭제 기능을 설정 가능
- **고급 검색 기능**: 문서의 내용을 토대로 파일 검색
- **다국어**: 한국어, 영어 선택 기능 구현
- **다크모드**: 다크모드 구현
- **반응형**: 반응형 UI 구현
